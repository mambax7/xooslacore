README FIRST
-----------------------

Xooslacore is a total rewrite of the WF-Resources module, it is smaller, faster will help in the aid of Xoosla Module development in the future
and providing many new features both for (Xoosla) developers and users alike. Automatic updates, stats on Xoosla modules and more.


Xoosla Candy was inspired by the Joomla modules, RokCandy by Rocket themes. Xooslacandy works in very much the same way but for Xoops.
This module was basically just a test for the new Xoosla module framework and I thought that it could come in handy for Xoopsers out there.
